<p><p>Public Cloud Non-Relational Databases, also known as NoSQL databases, come in several market types - Key Value Storage Database, Column Storage Database, Document Database, and Graph Database. Key Value Storage databases store data in a simple key-value pair format. Column Storage databases organize data in columns rather than rows for efficient querying. Document databases store data in a JSON-like format for flexible schema. Graph databases are designed for complex relationships between data points, making them ideal for social networks and recommendation engines.</p></p>
